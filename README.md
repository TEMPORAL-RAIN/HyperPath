## 🧠 What is HyperPath?

HyperPath is a **browser-history intelligence engine** that cross-references your real digital data — Brave/Chrome history, YouTube watch history, competition registrations, and AI session logs — to generate a zero-redundancy, evidence-backed learning or career roadmap.

Unlike generic "What should I learn?" tools, HyperPath **knows what you've already done** and skips it. It surfaces your hidden strengths, maps your real gaps, and points you to the exact next step.

> *"The roadmap is built from what you've actually done — not what you think you've done."*

## 🤖 AI Integration

Both profiles use the **Anthropic Claude API** (`claude-sonnet-4-20250514`) for real-time AI analysis.

### How it works

```javascript
const response = await fetch("https://api.anthropic.com/v1/messages", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({
    model: "claude-sonnet-4-20250514",
    max_tokens: 1000,
    system: USER_SPECIFIC_SYSTEM_PROMPT, // different per profile
    messages: [{ role: "user", content: userQuestion }]
  })
});
```

### AI Entry Points

| Where | What it does |
|-------|-------------|
| **DNA Profile Screen** | Analyses full digital footprint · 3 preset questions (strongest signal, 30-day focus, biggest gap) |
| **Roadmap Screen** | Explains why this sequence is optimal for this specific user |
| **Phase Modals** | Per-step AI: "Why is this right for me?" + "How long will this take me?" |
| **Dashboard Nudge** | Daily AI insight rotating from 4 personalised nudges |

Each user has a **completely separate system prompt** loaded with their specific data (visit counts, search terms, video titles, project names) so the AI's responses are genuinely personalised — not generic career advice.

## 🧩 How to Add a New Profile

Each user profile is a self-contained HTML file. To create a new one:

1. **Copy** `gaurav.html` (tech) or `vaibhav.html` (business) as your template
2. **Replace the data constants** at the top of the file:
   - `BRAVE_DATA` / `CHROME_DATA` — your browser export
   - `YT_DATA` — your YouTube watch history
   - `INTEL.buildFullProfile()` — your archetype + signals
3. **Update the AI system prompt** in `callAI()` with your specific data points
4. **Add your card** to `index.html` pointing to the new file

Data sources to export:
- **Chrome/Brave** → Settings → History → Export (use a history export extension)
- **YouTube** → Google Takeout → select YouTube → watch-history.json
